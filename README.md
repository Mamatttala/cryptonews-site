# CryptoNews Website

A complete, responsive, and accessible news website for cryptocurrency, finance, markets, and technology news.

## Project Structure

```
/cryptonews-website
├── index.html          (Home Page)
├── news.html           (News Listing Page)
├── blogs.html          (Blogs Listing Page)
├── contact.html        (Contact Page)
├── css/
│   └── style.css       (Master Stylesheet)
├── js/
│   └── main.js         (Main JavaScript File)
└── assets/
    └── images/         (Placeholder images)
```

## Features

1. **Responsive Design** - Works on mobile, tablet, and desktop devices
2. **Theme Switcher** - Light/Dark mode with localStorage persistence
3. **Multi-page Layout** - Home, News, Blogs, and Contact pages
4. **Interactive Elements** - Navigation menu, news filtering, contact form validation
5. **Modern UI** - Clean, professional design using CSS Grid and Flexbox

## Pages

- **Home Page (index.html)**: Featured article and latest news grid with sidebar
- **News Page (news.html)**: Filterable news articles by category
- **Blogs Page (blogs.html)**: Blog posts with author information
- **Contact Page (contact.html)**: Contact form with validation

## Technologies Used

- HTML5 Semantic Tags
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript ES6+ (Modules, DOM manipulation)
- Google Fonts (Inter)
- Picsum Photos for placeholder images

## How to Use

1. Open `index.html` in a web browser to view the home page
2. Navigate between pages using the top navigation menu
3. Toggle between light and dark themes using the switch in the header
4. On the News page, filter articles by category using the filter buttons
5. On the Contact page, submit the form to see validation in action

## Browser Support

The website works in all modern browsers that support:
- CSS Custom Properties
- CSS Grid and Flexbox
- ES6 JavaScript features
- localStorage API
